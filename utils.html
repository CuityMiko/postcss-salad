沙拉为你提供了CSS中常用代码片断的快捷声明，如清除浮动，文本超长溢出省略号、垂直居中、图片代替文字等等

矩形

我们可以通过 rect 属性来声明一个矩形。

{% highlight css %} rect: [width] [height] [background-color] {% endhighlight %} input: {% highlight css %} .rect-a { rect: 30px 50px #ff0; } .rect-b { rect: 30px * #ff0; } {% endhighlight %} output: {% highlight css %} .rect-a { width: 30px; height: 50px; background-color: #ff0; } .rect-b { width: 30px; background-color: #ff0; } {% endhighlight %}

圆形

我们可以通过 circle 属性来声明一个圆形。

{% highlight css %} circle: [diameter] [background-color] {% endhighlight %} input: {% highlight css %} .circle-a { circle: 50px #ff0; } .circle-b { circle: 50px *; } {% endhighlight %} output: {% highlight css %} .circle-a { width: 50px; height: 50px; border-radius: 50%; background-color: #ff0; } .circle-b { width: 50px; height: 50px; border-radius: 50%; } {% endhighlight %}

三角形

我们可以通过 triangle 属性来声明一个三角形。

{% highlight css %} triangle: [size] [direction] [color] {% endhighlight %} input: {% highlight css %} .triangle-a { triangle: 5px top #ff0; } {% endhighlight %} output: {% highlight css %} .triangle-a { display: inline-block; width: 0; height: 0; border: solid transparent; border-width: 5px; border-bottom-color: #ff0; } {% endhighlight %}
